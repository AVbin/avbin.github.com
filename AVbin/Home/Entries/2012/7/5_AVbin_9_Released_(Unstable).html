AVbin 9 Released (Edited)

 

I announce AVbin 9 today with mixed feelings.  It marks a lot of good work in build scripts and switching to Libav.  On the other hand, since I wanted to minimize C code changes for this release, the branch of Libav (0.6) that we used ends up being _older_ code than we used in the 8.1 release -- with some regressions in quality/performance.


End users are encouraged to use the 8.1 release for now.


Developers are encouraged to test version 9 out and give us feedback.


API Changes:


  1. -DEPRECATED avbin_get_ffmpeg_revision()

  2. -ADDED avbin_get_libav_commit()

  3. -ADDED avbin_get_libav_version()


Edit: For the purposes of publicly usable release versions, we’re just going to pretend version 9 didn’t happen.  It was a necessary and useful part of the development process, but just don’t use it.  Use version 10 or later!

Thursday, July 5, 2012
